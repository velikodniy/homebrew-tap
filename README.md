# Vadim's Homebrew Taps

## How do I install these formulae?

`brew install velikodniy/tap/<formula>`

Or `brew tap velikodniy/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "velikodniy/tap"
brew "<formula>"
```
